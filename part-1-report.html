Data Visualization Report

Summary:

This report contains 5 plots generated by ggplot R in total including bar plot, box plot, jitter plot, and histogram plot. This report is focusing on analyzing thrombosis levels in different variables such as age, gender, protein, and cholesterol values. Readers can get an overview of relationship between variables and thrombosis levels as well as the distribution of the thrombosis itself through the static plots.

 

Dataset: Thrombosis Data

This dataset contains statistics for different examinations and corresponding thrombosis levels. It includes 15 variables and 937 rows. Thrombosis is one of the major diseases that people concern about in a worldwide range. This cleaned dataset is created for providing a better base to analyze the thrombosis and corresponding variables to predict and decrease disease incidence. There are 6 related questions raised as the following to better guide readers going through.

 

The data is named as: Thrombosis.csv

 

Initial Analysis Questions:

1. How gender related to thrombosis?

2. What is the relationship between age and thrombosis?

3. Any difference in thrombosis by Pattern Observed in ANA Examination?

4. How is the distribution of Protein Observed in Laboratory examinations?

5. How is the distribution of Cholesterol Observed in Laboratory examinations?

6. What is the Bilirubin distribution Observed in Laboratory examination?

 

 

Discoveries & Insights:

 

Figure 1: Bar Plot

 

 

Comment: This is a bar plot focusing on the relationship between gender and the level of thrombosis. It was created since bar plot can display multiple categories, summarize vital values, and easily compare larger changes among different groups. In this bar plot, it shows that for both males and females, thrombosis is mostly in level 0 which means negative. Compared to males, females are more sensitive to thrombosis, females both have a high population in negative and positive thrombosis. What’s more interesting is that the bar plot shows the total population of missing values in gender; readers can easily see the missing values are only range from level 0 (negative) to level 1 (positive). Moreover, this plot shows an overview that the most tested people are negative and the number of populations decrease when the level of thrombosis increases.

 

Figure 2: Box Plot

Comment: This is a box plot on indicating the total difference in each level of thrombosis and age by exploring the maximum and minimum number of populations in ages. From this plot, more detailed statistical information such as mean, median and range which can be used to compared one variable to the others. For example, we can see that although the mean age of females and males are very similar when test positive thrombosis, males have a lower 1st quarter age in around 20 while females possess a high 3rd quarter age in around 50. When Thrombosis is in level 0 (negative), males have a wider range in ages while in level 2, only females are validated to be very positive. This shows an overall increase in range to females with the increasing of thrombosis levels.

 

Figure 3: Jitter Plot

 

 

Comment: This is a jitter plot in discovering difference in thrombosis by Pattern Observed in ANA Examination. The jitter plot is chosen because different like scatter plot which is better in showing the relationship between 2 variables, jitter plot can show the distribution of data by dispersing points. Also, it better visualizes the density of the data and the relationship between the thrombosis level and ANA Pattern as well as finding clusters in the data. From this plot, ANA Patterns of P and S have high densities distribution. The density is heavy especially when testing level 0 (negative). It also shows the distribution of missing ANA Patterns which normally clustering in level 0 and bad values of thrombosis in level 3. Compared to level 2(very positive), level 1(positive) have more ANA Patterns observed in examination.

 

 

Figure 4: Histogram Plot

 

Comment: This is a histogram plot giving insight on Protein distributions among levels of thrombosis. The histogram is chosen because it can show a frequency distribution by means. Different like bar plot, histogram can help determine whether the output of the examination is distributed normally. The fitting line also helps us to get an overview of changes smoothly. The histogram plot communicates that protein have a very perfect bell-shape distribution when test negative with the mean value around 7. Compared to level 0, level 1, 2, and 3 all shows a little bit right skewed distribution seems that a lower protein value has a strong relationship with a positive thrombosis examination. What’s interesting is that for both level 2 and 3, there is a gap when protein = 7.5 and a double peaked pattern then protein = 8, which might explain as the least likely protein value when thrombosis is positive.

 

Figure 5: Histogram Plot

 

Comment: This histogram is built with the goal to compare the difference between the distribution of protein values and cholesterol values. For this histogram, we can have an overview of the frequency of the cholesterol values and comparison with protein. Using the blue color for cholesterol distribution, it shows a similar bimodal for level 2 and 3 to protein values. However, it shows a little bit of right skewed when test negative for cholesterol distribution. Seems that the mean cholesterol value is the same (around 200) when thrombosis is around 0 to 1. Also, it shows a little bit right skewed pattern when the level is 0 or 1. Moreover, the histogram shows bad values when level = 3 with an extremely high frequency of 150 and 210 cholesterol which can be sifted out to provide a clear overview.

 

Figure 6: Histogram Plot

Comment: This histogram is built to compare the distribution of bilirubin to the distributions of cholesterol and protein. From the fitting line and the histogram, we can see very obvious right skewed patterns. For level 0 to 2, the distributions reach a peak when bilirubin is around 0.5 and decrease dramatically until reaching 1. There are also slight changes for level 1 and 2 because when bilirubin is around 1, the distributions show fluctuations and even a double peaked pattern for level 3. The histogram shows a specific range of bilirubin which can help scientist to predict and prevent future disease incidence.
